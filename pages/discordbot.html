GooseBot Discord Bot

Project Overview

    Project Discord Bot using discord.py libraries to implement commands. First time learning API Wrappers and programming within one

    Features

  • Logs discord messages, includes timestamp, user, channel, and message content.
  • Embeds images onto discord natively
  • Embeds Native Discord Buttons
  • Serves 400+ Members over 5 Discord Servers

Technologies Used

  • discord.py API Wrapper
  • python-dotenv virtual environment library
  • requests, os, random
  • GitHub for storing and managing code, facilitating collaboration and version control for project

Commands

  • These Commands are fun and most include simple images when used
  • !dinner
  • !spaghetti
  • !kill
  • !goose
  • !pfp @user
  • !help
