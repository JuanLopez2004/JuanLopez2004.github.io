Web Commerce Site

Project Overview

    In a group of 5 for CSCI 467 Software Engineering, we created a website that hooks up to Dr. Ege's Parts Database and Credit Card API. The website allows users to search for parts, add them to a cart, and checkout using a credit card. The website also has a product catalog that allows users to browse through all the products available. The website is built using ReactJS for the frontend and NodeJS/ExpressJS for the backend.

    Contributions

  • Created multiple webpages
  • Created a form for checkout page and checkout page frontend
  • Supported backend groupmate with NodeJS/ExpressJS pair programming
  • Contributed to web page design
  • Contributed CSS design
  • Created pagination for product catalog

Technologies Used

  • ReactJS for frontend
  • HTML for layout
  • CSS for design
  • SQLite for databasing
  • ExpressJS for Credit Card API & Parts Lookup
  • GitHub for storing and managing code, facilitating collaboration and version control for project
