Project Overview

This Android application is a simple yet effective To-Do List Checker built using Java in Android Studio (with the Ladybug plugin). It allows users to manage daily tasks efficiently with persistent storage and interactive checkboxes.

Technologies Used

  • Java
  • Android Studio (Ladybug)
  • SQLite for persistence
